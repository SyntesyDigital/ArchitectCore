# ArchitectCore

## Descripción

ArchitectCore contiene la funcionalidad basica para la implementación del CMS : 
- Tipologias
- Paginas con Page Builder
- Campos dinamicos
- Categorias y tags
- Interfaz

## Instalación

php artisan architect:install
